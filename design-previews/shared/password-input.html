PasswordInput

Shared controlled input with independent show/hide toggle. Used by both Registration and Login forms.

Masked (default)

Revealed

With error

Password must be at least 8 characters.

Disabled (submitting)

Behaviour invariants

  • Each field's toggle is fully independent — toggling one does not affect the other.
  • Default state: masked (type="password").
  • Toggle is type="button" — never submits the form.
  • aria-label changes: "Show password" ↔ "Hide password".
  • Toggling does not move focus or clear the value.
